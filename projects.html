Secure Login System

This project is a Python-based secure login and registration system designed with strong authentication practices and administrative functionality. It uses bcrypt hashing for password storage, enforces strong password requirements, and implements progressive account lockout protection to defend against brute-force attacks.

Maze Generator & Solver

This project is a Java-based application that creates random mazes using a depth-first search (DFS) algorithm. Users input their own maze dimensions and can create a maze manually, or generate one automatically. The program then solves the maze, clearly displaying the solution path for the user.

Steganography Tool

A Python-based steganography application with a simple Tkinter GUI that allows users to hide secret messages inside PNG images and retrieve them later. The tool uses least significant bit (LSB) encoding to embed data at the pixel level while keeping the image visually unchanged.
View on GitHub
